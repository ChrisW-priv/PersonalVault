Proszę wybrać metodę uwierzytelniania.

WEiTI LDAP

Uwierzytelnianie przy pomocy kont wydziałowych z maszyn elektron i mion. Metoda ta dostępna jest tylko dla osób, które mają założone konta wydziałowe WEiTI.
WEiTI klucze

Uwierzytelnianie przy pomocy prywatnych kluczy wygenerowanych wcześniej. Metoda ta pozwala wygenerować klucz ważny przez semestr, który następnie można zapisać w aplikacji (kliencie pocztowym, programie kalndarza obsługującym CalDAV, itp.) Dzięki temu aplikacja ma dostęp do danych a jednocześnie nie musi znać właściwego hasła użytkownika.
USOS API

Uwierzytelnianie przy pomocy kont uczelnianych z wykorzystaniem USOS API. Metoda ta oraz metoda USOS CAS są szczegónie przydatne dla osób nie posiadających kont wydziałowych. Metody te można wykorzystywać zamiennie są alternatywnymi sposobami wykorzystania tych samych danych autoryzacyjnych.
USOS CAS

Uwierzytelnianie przy pomocy kont uczelnianych z wykorzystaniem centralnego serwera autoryzacyjnego CAS PW. Metoda ta oraz metoda USOS API są szczegónie przydatne dla osób nie posiadających kont wydziałowych. Metody te można wykorzystywać zamiennie są alternatywnymi sposobami wykorzystania tych samych danych autoryzacyjnych.
